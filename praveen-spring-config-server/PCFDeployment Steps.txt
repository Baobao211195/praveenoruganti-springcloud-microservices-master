How to Deploy your springboot application in PCF(Pivotal Cloud Foundary)?

1.Install the CF CLI(Cloud Foundry Command Line Interface)

https://cli.run.pivotal.io/stable?release=windows64&source=github

2.Go to the springboot application folder which you need to deploy in PCF and make sure manifest.yml is available.

For example,
cd D:\Praveen\Projects\GITworkspace\springboot-cloud-eureka-server-master\praveen-sping-cloud-eureka-server

Example manifest.yml for spring boot application praveen-sping-cloud-eureka-server

---
applications:
 - name: praveen-sping-cloud-eureka-server
   path: target/praveen-sping-cloud-eureka-server-1.0.jar
   domain: cfapps.io
   memory: 512m
   instances: 1

3.Deploy the above springboot application in PCF by running the below command

cf push

4.For scaling the memory if app is crashed after deploying in PCF

cf scale app-name -m 1G

for example,

cf scale praveen-sping-cloud-eureka-server -m 1G

5.For deleting app in PCF

cf delete -r app-name

for example,

cf delete -r praveen-sping-cloud-eureka-server


Please note you need to register for free account in PCF using https://account.run.pivotal.io/sign-up
